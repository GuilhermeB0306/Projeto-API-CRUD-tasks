# 📋 Task Manager API

Aplicação Full Stack para gerenciamento de tarefas (To-Do), desenvolvida com Node.js e Express, permitindo a organização e manipulação de tarefas através de uma interface integrada com o backend.

---

## 🚀 Tecnologias utilizadas

- Node.js  
- Express  
- JavaScript  
- HTML, CSS e JavaScript (Frontend)  
- API para comunicação entre frontend e backend  

---

## 📌 Funcionalidades atuais

- Listagem de tarefas  
- Atualização de tarefas  
- Remoção de tarefas  
- Integração entre frontend e backend  
- Estrutura organizada em padrão MVC  

---

## ⚠️ Funcionalidades em desenvolvimento

Este projeto ainda está em evolução. As próximas melhorias planejadas são:

- ➕ Implementação de botão para criação de novas tarefas  
- 🔄 Funcionalidade para alterar o status das tarefas (ex: “a fazer”, “fazendo”, “concluído”)  
- 🎯 Melhorias na interface e experiência do usuário (UI/UX)  
- 🧠 Validação de dados no backend  

---

## 📁 Estrutura do projeto

- frontend/ → Interface do usuário  
- src/ → Backend da aplicação  
  - controllers/ → Lógica das rotas  
  - models/ → Estrutura dos dados  
  - services/ → Conexão com banco de dados  
  - routes/ → Definição dos endpoints  
- server.js → Inicialização do servidor  
- .env → Variáveis de ambiente
